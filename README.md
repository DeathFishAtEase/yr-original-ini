《尤里的复仇》原版 INI 文件存档
 ---

用于方便阅读 `1.000` 版本与 `1.001` 版本的差异项

以便满足基于 `1.000` 制作的 Mod 作品同步更新需求

原版 `1.000` 版本取自 `ra2md.mix\localmd.mix`

原版 `1.001` 版本取自 `expandmd01.mix`，发生变更的文件如下
 - `rulesmd.ini`
 - `soundmd.ini`

[1.001 版本更新内容差异对比页面入口](https://github.com/DeathFishAtEase/yr-original-ini/pull/1)
